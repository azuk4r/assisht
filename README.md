# setup
### install
**clone the repo:**
```
git clone https://github.com/azuk4r/assisht
```
**go to the folder:**
```
cd assisht
```
**and then install with:**
```
sh install.sh
```
**all in one:**
```
git clone https://github.com/azuk4r/assisht && cd assisht && sh install.sh
```
### uninstall
**to remove all config files:** 
```
sh uninstall.sh
```
